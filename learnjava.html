Introduction to Java

Java was developed by a team led by James at Sun Microsystems. Originally called Oak, it was designed in 1991 for use in embedded chips in consumer electronic appliances.

In 1995, renamed Java, it was redesigned for developing Internet applications.

Java has become enormously popular. Its rapid rise and wide acceptance can be traced to its design characteristics, particularly its promise that you can write a program once and run it anywhere

For the history of Java, see java.sun.com/features/1998/05/birthday.html.
